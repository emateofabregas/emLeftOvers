# emLeftOvers
Gizmo that calculates the residual pixels between beauty and light groups when you're doing CG compositing adjusting the LGT using AOV Grade or a substractive setup.
![emLeftOvers_Cover](https://github.com/user-attachments/assets/8a62d8b2-8cbe-4f0b-90a8-5e89738374e1)
Thank you Robert Kouts for your render from his shortfilm 'Iron Age' for this demo. 
![emLeftOvers_gizmo](https://github.com/user-attachments/assets/084ae0ae-c16c-4006-9e7a-a47f61f80d03)
## Features
* Checbox to check the leftovers pixels.
* string knob for the prefix of LGT.
* string knob for the prefix of emission.


![emLeftOvers_Knobs](https://github.com/user-attachments/assets/f6ee87f1-6d1d-422f-8bb8-d39e1d6dad2c)

## How to use it
1. You have to connect the tool after your LGT work comp using 'Img' one.
2. Connect your raw render into 'Raw Render Input'.
3. Write the LGT and emission prefix.
4. Check the chebox thing and increase the exposure of your render to see if the leftovers are created well.
