# emLeftOvers
Gizmo that calculates the residual pixels between beauty and light groups. 
![emLeftOvers_Cover](https://github.com/user-attachments/assets/8a62d8b2-8cbe-4f0b-90a8-5e89738374e1)
Thank you Robert Kouts for your render from his shortfilm 'Iron Age' for this demo. 
![emLeftOvers_gizmo](https://github.com/user-attachments/assets/084ae0ae-c16c-4006-9e7a-a47f61f80d03)
## Features
* Checbox to check the leftovers pixels.
* string knob for the prefix of LGT.
* string knob for the prefix of emission.
